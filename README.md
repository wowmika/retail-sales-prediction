# Retail Sales Prediction

## Problem
Rossmann operates 3,000+ drug stores across Europe. 
This project predicts daily sales up to 6 weeks in advance.

## Models Used
- Linear Regression
- Lasso Regression
- Decision Tree
- Random Forest ✅ (Best Model - MAPE 5.65%)

## Libraries
Python, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib

## Result
Random Forest achieved lowest MAPE of 5.65%
Top features: Customers, CompetitionDistance, StoreType, Promo
